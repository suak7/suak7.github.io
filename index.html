Projects

  • Custom 32-bit x86 Bootloader with EHCI (USB 2.0) Driver Support: I developed a custom 32-bit x86 bootloader and a freestanding C kernel. A core feature of this project was the implementation of an EHCI driver, which enabled bare-metal USB 2.0 device enumeration through control transfer protocols and UTF-16 string descriptor parsing.
  • Custom 64-bit x86 Bootloader with AHCI (SATA) Storage Driver Support: This project involved engineering a bare-metal 64-bit x86 bootloader specifically designed to support an AHCI (SATA) storage driver for direct disk access. I wrote a DMA-based command execution system utilizing AHCI command lists and Physical Region Descriptor Tables (PRDTs) while adhering to strict physical memory alignment constraints.
